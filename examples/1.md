【题目要求】

请按照这样的日期格式（xxxx-xx-xx）每日生成一个文件，
例如生成的文件名为2017-12-20.log， 
并且把磁盘的使用情况写到到这个文件中，
不用考虑cron，仅仅写脚本即可

【习题答案】

本题答案：

```shell
#! /bin/bash
d=`date +%F`
logfile=$d.log
df -h > $logfile
```

需求升级：
```shell
#!/bin/bash
d=`date +%F`
dir=/data/logs/disklog
if [ ! -d $dir ]
then
    mkdir -p $dir
fi
df -h > $dir/$d.log
find $dir/ -mtime +365 |xargs rm -f
```
